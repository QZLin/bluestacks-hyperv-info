# BlueStacks HyperV Info

## Install

* Install via pip,git

```shell
git clone [repo]
cd repo-name
pip install -e .
```

## Usage
* Show Help Info  
  `python -m bluestacksv -h`  
* Show BlueStacks HyperV Port (Default)  
  `python -m bluestacksv`
